nvm install 20 -> instala a versão 20
nvm install --lts -> instala a versão long-term support (versão recomendada)
nvm list
nvm use 18 -> use a 18 que ja está instalado na minha maquina
nvm uninstall 18 -> desinstala a versão 18
nvm alias default 18 -> define a versão 18 como padrão
nvm unalias default -> remove a versão padrão
nvm ls-remote -> lista todas as versões do node disponíveis para download
nvm install 18.12.1 -> instala uma versão específica do node
